AWS Config

  • Create a new aws config file in src/config/awsConfig.ts
import { S3, S3Client } from "@aws-sdk/client-s3";

let jsonSecret= {
 AWS_ACCESS_KEY_ID: process.env.AWS_ACCESS_KEY_ID;
 AWS_SECRET_ACCESS_KEY: process.env.AWS_SECRET_ACCESS_KEY;
 BUCKET_NAME: process.env.BUCKET_NAME;
};
let s3 = new S3Client({
     credentials: {
       accessKeyId: jsonSecret.AWS_ACCESS_KEY_ID,
       secretAccessKey: jsonSecret.AWS_SECRET_ACCESS_KEY,
     },

     region: "us-east-1",
});

export { s3, jsonSecret };
  • S3: This is a service client object for interacting with S3.
  • S3Client: A constructor for creating S3 client instances.
  • This code creates an object jsonSecret to store AWS credentials and bucket name. AWS_ACCESS_KEY_ID and AWS_SECRET_ACCESS_KEY are typically retrieved from the IAM User we create before. BUCKET_NAME is the name of your S3 Bucket.

Uploading image to s3 bucket

Create a new file in src/controllers/imageController.ts In this file, create a function uploadImage that handles uploading an image to an S3 bucket

import { Request, Response } from "express";
import { s3, jsonSecret } from "../config/awsConfig";
import {
  PutObjectCommand,
  ListObjectsV2Command,
  GetObjectCommand,
  DeleteObjectCommand,
} from "@aws-sdk/client-s3";
import { getSignedUrl } from "@aws-sdk/s3-request-presigner";
export const uploadImage = async (req: Request, res: Response) => {
  try {
    const file = req.file;

    if (!file) {
      return res.status(400).json({ message: "No file uploaded" });
    }

    const fileName = Date.now() + "_" + file.originalname;
    const fileContent = file.buffer;
    const putObjectRequest = new PutObjectCommand({
      Bucket: jsonSecret.BUCKET_NAME ? jsonSecret.BUCKET_NAME : "",
      Key: `images/${fileName}`,
      Body: fileContent,
      ContentType: file.mimetype,
    });

    await s3.send(putObjectRequest);

    const getObjectCommand = new GetObjectCommand({
      Bucket: jsonSecret.BUCKET_NAME,
      Key: `images/${fileName}`,
      ResponseContentDisposition: "inline",
    });
    const url = await getSignedUrl(s3, getObjectCommand, { expiresIn: 3600 });

    return res.json({
      message: "Image uploaded successfully",
      data: url,
    });
  } catch (error) {
    console.error(error);
    return res.status(500).json({ message: "Internal server error" });
  }
};
  • This code creates a PutObjectCommand object with details for uploading the file to S3. Bucket: The S3 bucket name retrieved from jsonSecret. Key: The path within the bucket where the image will be stored (prefixed with “images/”). Body: The file content retrieved from the buffer. ContentType: The MIME type of the uploaded file extracted from file.mimetype.
  • Uses the s3 client to send the PutObjectCommand and upload the image asynchronously. If the upload is successful, it creates a GetObjectCommand to retrieve the uploaded image from S3. Sets the ResponseContentDisposition header to “inline” to indicate the image should be displayed directly in the browser. Uses the getSignedUrl function to generate a temporary pre-signed URL for accessing the uploaded image. This URL expires after 3600 seconds (1 hour).
